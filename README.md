# SQL Sales Analysis Project

## Overview
This project analyzes a sales dataset using SQL to uncover key business insights and trends. The goal is to practice real-world data analysis skills and demonstrate SQL proficiency for data analyst roles.

---

## Objectives
- Identify top-selling products  
- Analyze sales performance by city and country  
- Track monthly and yearly sales trends  
- Find the best-performing year  
- Calculate Average Order Value (AOV)

---

## Tools & Technologies
- MySQL  
- SQL (Joins, Aggregations, Subqueries, CTEs)  
- GitHub for version control

---

## Dataset
The dataset contains sales transactions including:
- Order ID  
- Order Date  
- Product  
- City / Country  
- Sales amount

---

## Key Analysis Performed

### Top Products Analysis
Identified products generating the highest revenue.

### Sales by Location
Analyzed sales distribution across cities and countries.

### Monthly Sales Trend
Used date functions to analyze seasonality and monthly performance.

### Yearly Performance
Determined the highest-performing year based on total sales.

### Average Order Value (AOV)
Calculated AOV by aggregating order totals first.

---

## Sample Insights
- Certain products contribute significantly to revenue.  
- Sales show seasonal patterns across months.  
- A small number of cities generate a large portion of sales.
